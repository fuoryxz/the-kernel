This is a classic starting skeleton for a "write your own OS"
project (based on the OSDev wiki) — it actually boots via GRUB/QEMU and prints text to the screen
in protected mode. From here, you can build it up further: GDT/IDT, interrupt handlers, keyboard driver,
memory manager, a simple scheduler — but those are large, separate pieces.
Let me know if you want to move in any of those directions.
